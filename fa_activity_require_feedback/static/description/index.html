Activity Require Feedback

This module allows you to configure whether to ask for feedback after completing an activity.

You can configure requiring feedback on each type of activity

How to Use

Firstly, go to your Activity Types menu

Secondly, open activity type you want to require feedback, then tick to "Require Feedback"

Now, when done activity, user need to input his feedback first
